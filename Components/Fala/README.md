# Utilização
~~~
Para utilizar a componente Speak é necessário baixar o jar em: https://github.com/watson-developer-cloud/java-sdk/releases/download/java-sdk-7.0.0/ibm-watson-7.0.0-jar-with-dependencies.jar
e colocá-lo em seu build path. Depois, para usar a função speak, basta usar Speak.speak(texto), pois é uma função static.
~~~
